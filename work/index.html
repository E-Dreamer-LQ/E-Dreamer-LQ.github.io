项目实战

场景映射数据闭环

项目内容

通过传感器数据，建立真实世界到数字 3D 世界的映射，获取空间，语义，趋势等信息。

主要负责

实现4路avm摄像头+5路xpu摄像头+2路激光数据实时同步采集 。

项目成果

1.实现基于视频压缩的传感器数据高帧率采集；

2.实现基于数据前馈的解决多传感器时间同步的创新算法；

3.负责实现从采集到存储的完整数据链路，极致可视化，数据驱动算法；

4.技术涉及：a. 标定及时间同步；b.3d 点云网格算法；c. 多线程多进程加速；f. 实车调试经验；

立体感知传感器标定

项目内容

探索激光雷达标定新方法，简化标定流程并且减少标定精度影响因素，实现激光雷达标定自动化和激光雷达与相机的联合标定；

工作内容

探索并实现高效的激光雷达标定方法；

项目成果

a.熟悉并掌握传统激光雷达标定方法； b.开发了基于平面法向量的激光雷达标定方法；

数字孪生——“左脚走路”

项目内容

基于渲染与数字3D世界内容，生成关键场景数据，利用虚拟数据释放人工标注成本，解决感知训练“长尾”问题，提高core case精度。

工作内容

1.利用 GAN 实现虚拟数据与真实数据的风格迁移训练；

2.基于 faster rcnn 完成虚拟数据对真实数据训练 kpi 初步验证；

3.基于 UE5, 脱离 carla 束缚，建立虚拟仿真数据生成平台；

4.技术涉及：a.carla 引擎；b.UE5 开发；c.GAN 应用；d.faster rcnn；

原子化虚拟交通流设计——“右脚走路”

项目内容

基于车辆运动学模型，原子化交通流设计，配合 3d 可视化技术，真实路网上实现虚拟交通流。

工作内容

1.“棋局式”泛化——负责车辆场景设计；

2.实现 idm 跟车模型应用于原子化交通流功能；

3.负责 pure persuit 控制算法改善拐弯流畅性；

4.基于采样的变道超车等交通流设计探索；

5.技术涉及：a. 车辆运动学；b. idm 跟车模型；c. 规划控制；d. 场景泛化设计；f. 碰撞检测

xbrain——无人配送集群小车项目

项目内容

基于数据闭环，目标打造一款场景覆盖度高，可靠性高，成本有充分竞争力的自主出行智能系统，建立自动驾驶集群小车，实现 A 到 B 的自主出行。目前已实现重定位导航、轨迹跟踪、速度规划、绕障。

工作内容

  1. 从 python 到 cpp(ros2) 的环境迁移，设计并建立传感器数据、定位、规划、控制的闭环；
  2. 参与 lane 级导航功能实现，基于 A* 算法的 A 到 B 的最短路径规划；
  3. 负责基于 lqr 和 pure pursuit 算法的融合传感器数据信息避障循迹功能实现；
  4. 负责纵向速度规划，实现入弯慢，出弯快的智能纵向控制；
  5. 基于 lattice 和 costmap 的绕障、变道等功能持续实现中；
  6. 技术涉及：a. 底盘调试；b. 车辆运动学；c. 控制算法；d. 局部规划及运动规划；f. 智能驾驶闭环；
